# 天地劫手游助手

# 功能
1. 自动完成天地劫手游中的每日任务
2. 自动完成好友任务
3. 自动完成天地阁任务
4. 持续更新，有兴趣可以自己下载源码学习。

# 详细说明
1. tdjgame_path文件为为天地劫程序所在路径，需要修改为自己的实际安装目录
2. tdjimages为天地劫对应图标截图 如果不准确可自己替换
3. 首次使用启动游戏，需手动登录游戏一次来记录账号信息
4. 此工具运行在win10系统，游戏客户端也是win10不是运行在模拟器上
